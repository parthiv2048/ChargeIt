# ChargeIt

## About
ChargeIt is an iOS app that finds nearby charging stations for your electric vehicle

## Tech Stack
- Swift
- UIKit
- Storyboards
- XCode version 26.2

## Features
The UI has been created using Storyboards, but the functionality is yet to be implemented. The following UI elements have been added -
- Create an account with your email, name, and phone number
- Sign in using a created account, Google, or Facebook
- Use Forgot Password to reset your account password
- During onboarding, optionally add your electric vehicle type and payment information for convenient searching and use
- On the home screen, which displays a map of nearby charging stations, search for specific stations or filter by power and port type
